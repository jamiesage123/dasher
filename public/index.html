About Dasher

Welcome to Dasher!

Dasher is a customisable dashboard built to display various pieces of information that may be of use! Dasher will automatically update as long as the server is still available, with everything updating every 15 minutes to ensure you have the latest information!

Created with by up772629

What can be customised?

  • House colours can be changed (to whatever you want!)
  • Background Image Category [Nature, Buildings, Food, People, Technology, Objects]
  • Displaying and Hiding specific widgets
  • Type whatever you want in your notepad!

How can I customise Dasher?

Customisation is easy! All you have to do is click on the button on the left side, it'll open up a menu, then just follow all the steps displayed! Easy as pie!

We wanted to say Thank You!

Dasher makes use of various different public systems to get all your data, thanks for making stuff available!

  • Weather API is provided by DarkSky [ https://darksky.net ]
  • Location API is provided by Google [ https://google.com ]
  • Weather Icons are provided by erikflowers [ http://weathericons.io ]
  • Latest Headlines are provided by News API [ https://newsapi.org ] and BBC News [ https://bbc.co.uk/news ]
